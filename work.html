I am currently on leave from Yale doing research with the Yale Efficient Computing Lab under Professor Lin Zhong, Ph.D., to develop integration techniques for cell phones and flying drones in order to realize mobile hands-free computing experiences.

My tasks have included:

  • Writing a framework for stabilizing text on the phone display
    • Modifying the native Android graphics pipeline and display (DRM) driver
  • Developing a control interface between PC, Android, and an STM32 board using Wifi Direct and USB
    • Modifying the Crazyflie nanocopter firmware to send and receive control packets via USB
  • Using OpenCV and pre-trained neural networks to create a computer vision system for human recognition, mapping, localization, and displacement calculation using a single camera
  • Troubleshooting and optimizing the drone's power consumption and size
